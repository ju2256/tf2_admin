This Zip File is part of 
    'How-To: Create a TF2 chalkboard style MOTD with HTML' 
originally found on 
     http://www.specialattack.net/node/39
     
To install the MOTD on your server, follow these steps:

*) Make changes to the included motd.png
*) Change the URLs to the images in motd.html to point to your webserver
*) Upload the motd.html, motd.png and blackboard.png to your webserver
*) Create a motd.txt in your gameserver's 'tf' directory, and insert a link to you uploaded html file in the first line (e.g. http://yoursite.com/motd.html)

That's it.
